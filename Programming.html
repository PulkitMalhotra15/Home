SPYDER

Scientific Python Development Environment

Spyder is an open source cross-platform integrated development environment (IDE) for scientific programming in the Python language.
Spyder integrates with a number of prominent packages in the scientific Python stack, including NumPy, SciPy, Matplotlib, pandas, IPython, SymPy and Cython, as well as other open source software.

As part of my research project at University of Mumbai, I learnt Spyder.

  • It was used to analyse lunar dust under various spectrums.
  • We used further processed all the images to be fit for analysis, via cropping, rotating and centering of the moon
  • Created a routines to further plot data under the various spectrums and save them in a data matrix
  • We thus created a model to predict and explain the behaviour
  • I particularly used Aperture Photometry (photutils.aperture) to analyse Lunar dust data in my research internship at University of Mumbai
